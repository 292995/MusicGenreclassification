# Models Directory

## Saved Models
- `music_genre_model.h5` - Trained neural network (not in repo)
- `label_encoder.pkl` - Label encoder for genre names (not in repo)

## Model Performance
- Accuracy: 80%
- Features: 40-dimensional feature vector
- Classes: 10 music genres

## Training Parameters
- Optimizer: Adam
- Loss: Sparse categorical crossentropy
- Epochs: 100
- Batch size: 32
